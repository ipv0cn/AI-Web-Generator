# 前端美化方案

## 设计方向

**主题**：AI Web Generator — URL 即 Prompt，即时生成网页。
**核心隐喻**：搜索栏是一个"传送门"——输入 URL，AI 即时为你生成一个全新的网页。
**受众**：开发者、技术爱好者、对 AI 能力好奇的用户。

## 设计决策

### 色彩体系（摆脱默认模板）

放弃常见的"深色背景+紫蓝渐变"模板。选用更独特、更有"数字生成"感的配色：

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `#0a0a14` | 深邃碳蓝黑底，不纯黑，有呼吸感 |
| 主色 | `#00d4aa` | 青绿色——代表 AI/数字/新鲜感 |
| 强调色 | `#f5a623` | 暖琥珀——用于按钮，象征"创造/火花" |
| 表面 | `#141428` | 半透明深色卡片/输入框背景 |
| 文字 | `#d0d0e0` | 浅灰紫色正文 |
| 标题 | `#f0f0f8` | 近乎白色，用于大标题 |

### 字体（系统字体限制）

由于用户要求"无外部依赖"，使用系统字体组合：

| 角色 | 字体栈 |
|------|--------|
| 标题 | `system-ui, -apple-system, sans-serif`，weight 200，letter-spacing 加宽 |
| 输入框 | `"SF Mono", "Fira Code", monospace`（保留现有） |
| 正文/说明 | `system-ui, -apple-system, sans-serif` |

### 布局

单栏居中，垂直层次：
```
┌────────────────────────┐
│         [SVG Logo]       │  ← 简洁 AI 图标
│   AI Web Generator       │  ← 大标题，细体加宽字距
│   URL即Prompt，即时生成   │  ← 副标题
│                          │
│ ┌──────────────────────┐ │
│ │ /baidu.com      [→] │ │  ← 玻璃拟态输入框 + 琥珀按钮
│ └──────────────────────┘ │
│                          │
│  [baidu.com] [github] [google]  ← 示例卡片，可点击填充
│                          │
│     Powered by AI · 即时生成   ← 页脚
└────────────────────────┘
```

###  Signature 元素（唯一的视觉记忆点）

**琥珀按钮的脉冲光晕** — 当输入框聚焦或悬停于按钮时，按钮周围出现一圈缓慢脉动的光晕环（纯 CSS `@keyframes`）。这个动效隐喻"生成/创造"的行为。其他元素保持安静收敛。

## 具体改动项

### 1. 背景：渐变动画（替换纯色）
- 从 `#0a0a14` 缓慢过渡到 `#0d1520` 再回到 `#0a0a14`
- 纯 CSS `@keyframes`，12s 循环，`background` 属性过渡
- 仅轻微变化，不分散注意力

### 2. SVG Logo
- 简约的"抽象无限/网络环"图形，双色渐变（青绿→琥珀）
- 约 48×48px，位于标题上方，居中

### 3. 标题 + 副标题
- 标题："AI Web Generator"——`font-weight: 200`，`letter-spacing: 0.08em`，`font-size: clamp(1.8rem, 4vw, 2.8rem)`
- 副标题："URL 即 Prompt — 输入目标站点，AI 即时生成"——`color: #8888a0`，`font-size: 0.95rem`

### 4. 搜索栏（玻璃拟态 + 签名动效）
- 背景：`rgba(255,255,255,0.04)` with `backdrop-filter: blur(12px)`
- 边框：`1px solid rgba(255,255,255,0.08)`，聚焦时变为 `rgba(0,212,170,0.3)`
- 输入框获得焦点时，按钮出现脉冲光晕环动画（Signature 元素）
- 输入框内右侧显示 `↵ Enter` 提示（浅灰色）

### 5. 示例快捷卡片
- 三个可点击的小卡片：`/baidu.com` | `/github.com/login` | `/google.com/search`
- 点击自动填入输入框（不跳转）
- 卡片样式：深色背景 `rgba(255,255,255,0.03)`，边框 `1px solid rgba(255,255,255,0.06)`
- hover：边框变为青绿色，轻微上移 `translateY(-2px)`

### 6. 加载动画
- 点击"前往"后，按钮变为仅显示 spinner（纯 CSS 圆环旋转）
- 按钮下方显示 "AI 生成中..." 文字
- 页面跳转时自然消失，无需 JS 控制

### 7. 页脚
- 小字居中：`Powered by AI · 每次访问由 AI 即时生成`
- 颜色：`#555568`，字号 `0.75rem`
- 不抢眼，仅作为收束

## 约束
- 保持单文件（`index.html`），所有 CSS/JS 内联
- 无外部依赖（无 npm、无 CDN 图标库、无 Google Fonts）
- 不改动 `app.py` 后端
- 不添加复杂交互动效，不引入框架

## 实现顺序

1. 背景渐变动画
2. SVG Logo
3. 标题 + 副标题
4. 搜索栏（玻璃拟态 + 脉冲光晕签名动效）
5. 示例卡片
6. 加载动画
7. 页脚
